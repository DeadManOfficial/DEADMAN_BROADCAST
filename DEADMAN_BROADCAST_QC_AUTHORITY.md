## PHASE 1: VISUAL FORENSICS CHECKLIST

### 1.1 ANALOG DEGRADATION AUDIT (CRITICAL)

**SCANLINES:**
```
[ ] Horizontal 480i scanlines visible (2px spacing)
[ ] Scanlines are consistent, not random digital noise
[ ] Opacity 3-5% - visible but not overwhelming
[ ] Applied across ENTIRE frame, not just portions

FAIL CONDITIONS:
- Clean digital look with no scanlines
- Vertical scanlines (incorrect format)
- Inconsistent/flickering scanline pattern
```

**VHS ARTIFACTS:**
```
[ ] Chroma noise/color bleed present (especially on reds)
[ ] Edge ghosting on motion trails
[ ] Tracking errors on transitions (if applicable)
[ ] Oversaturated magenta-cyan color shift visible
[ ] Tape wear texture (subtle warping)

FAIL CONDITIONS:
- Perfect color separation (too clean)
- No color bleed on red ON AIR lights
- Motion is too sharp/clean (needs analog blur)
```

**FILM GRAIN:**
```
[ ] 20-25% opacity grain texture present
[ ] Grain is organic, not digital noise
[ ] 35mm scan quality (fine, not chunky)
[ ] Overlay blend mode (not multiply/screen)

FAIL CONDITIONS:
- No grain (sterile digital look)
- Heavy digital noise (wrong type)
- Grain too uniform (needs organic variation)
```

**VIGNETTE:**
```
[ ] 15-20% strength vignette present
[ ] Soft oval falloff (not hard circular)
[ ] Center-weighted, gradual
[ ] Aids focus on subject

FAIL CONDITIONS:
- No vignette (frame too open)
- Heavy/dramatic vignette (too stylized)
- Square vignette (wrong shape)
```

**CHROMATIC ABERRATION:**
```
[ ] 1-2px RGB split on frame edges
[ ] Subtle, not pronounced
[ ] Appropriate for analog lens distortion

FAIL CONDITIONS:
- No chromatic aberration (too perfect)
- Heavy/stylized CA (looks like Instagram filter)
```

---

### 1.2 COLOR SCIENCE AUDIT

**PALETTE COMPLIANCE:**
```
[ ] Signal Red #FF2020 present (ON AIR/danger elements)
[ ] Phosphor Green #33FF33 visible (CRT text/data)
[ ] Cyan Blue #00FFFF used (monitor spill/accents)
[ ] Void Black #0A0A0A dominates shadows (deep, impenetrable)

FAIL CONDITIONS:
- Warm, inviting colors (wrong tone)
- Saturated natural skin tones (too healthy)
- Bright whites without cyan contamination
- reds that aren't punched to 120% saturation
```

**DESATURATION CHECK:**
```
[ ] Global -30% desaturation applied (except reds)
[ ] Skin tones have slight green undertone (sickly)
[ ] Never warm or inviting
[ ] Cold, institutional, processed

FAIL CONDITIONS:
- Natural/warm skin tones
- Full saturation on non-critical elements
- Inviting, cozy atmosphere
```

**COLOR GRADE VERIFICATION:**
```
[ ] Blacks crushed to 5 (not pure black)
[ ] Highlights clipped to 250 (not blown)
[ ] Shadows push cyan-teal (+15 cyan)
[ ] Midtones slight green shift (-5 magenta)
[ ] Highlights push cold blue (+10 blue)

FAIL CONDITIONS:
- Neutral color grade (no temperature push)
- Warm shadows (wrong color science)
- Natural midtones (lacks institutional feel)
```

---

### 1.3 LIGHTING AUDIT

**PRACTICAL SOURCES ONLY:**
```
[ ] All lights are motivated (CRT, tally, fluorescent, desk lamp)
[ ] NO beauty lighting/soft boxes visible
[ ] 20:1 contrast ratio minimum verified
[ ] Deep shadows are impenetrable black
[ ] Light sources match 1987 technology era

FAIL CONDITIONS:
- Soft, flattering light on subjects
- Even exposure (too broadcast-safe)
- Bounce cards or reflectors evident
- Modern LED characteristics
- Warm key light (unless tungsten desk lamp)
```

**CONTRAST VERIFICATION:**
```
[ ] Measured contrast ratio: ___:1 (minimum 20:1)
[ ] Shadows contain NO detail (void black)
[ ] Highlights don't blow (retain texture)
[ ] Chiaroscuro effect appropriate

FAIL CONDITIONS:
- Contrast ratio under 15:1
- Shadow detail visible (not threatening enough)
- Flat, even lighting
```

---

### 1.4 CAMERA/PHYSICS AUDIT

**MOTION CHARACTERISTICS:**
```
[ ] Frame rate is 24fps cinema (NOT 30fps or 60fps)
[ ] Camera movement is motivated and specific
[ ] If static: Imperceptible organic drift (not locked perfect)
[ ] If moving: Speed specified in ft/sec
[ ] No AI-smooth floating camera moves

FAIL CONDITIONS:
- 60fps "soap opera effect"
- Unnaturally smooth gimbal moves
- Perfect lock-off (too digital)
- Floaty, weightless camera drift
```

**SUBJECT PHYSICS:**
```
[ ] Objects have weight and momentum
[ ] Motion obeys gravity and inertia
[ ] No morphing or shape-shifting
[ ] No teleporting objects
[ ] Temporal continuity maintained

FAIL CONDITIONS:
- Weightless, floaty objects
- Teleporting props
- Morphing/melting (unless intentional glitch)
- Sudden disappearances
```

---

### 1.5 COMPOSITION AUDIT

**SHOT INTEGRITY:**
```
[ ] Subject clearly defined and identifiable
[ ] Action is specific and motivated
[ ] Environment supports narrative
[ ] No generic corporate video look
[ ] Hyper-specific details present (not vague)

FAIL CONDITIONS:
- Generic office with no distinguishing features
- Vague actions ("person works at desk")
- Could be any corporate video
- Lacks unique DEADMAN identifiers
```

## PHASE 2: ACOUSTIC FORENSICS CHECKLIST

### 2.1 MANDATORY AUDIO ELEMENTS

**60Hz HUM (NON-NEGOTIABLE):**
```
[ ] 60Hz electrical hum present at -40dB (or specified level)
[ ] Continuous throughout entire clip
[ ] Sine wave, not digital buzz
[ ] Provides "always broadcasting" foundation

KILL CONDITIONS:
- No 60Hz hum present
- Hum is too loud/intrusive
- Hum is digital/synthetic (wrong character)
- Hum cuts out mid-clip
```

**ANALOG STATIC TEXTURE:**
```
[ ] Subtle analog static/tape hiss at -50dB
[ ] Present throughout, not intermittent
[ ] Authentic tape character, not white noise
[ ] Adds analog warmth

KILL CONDITIONS:
- Clean digital silence
- No tape texture/hiss
- Digital artifacts instead of analog
```

---

### 2.2 FOLEY/SFX SYNC AUDIT

**TRANSIENT SYNC (CRITICAL):**
```
[ ] Hard effects sync within 2 frames of visual impact
    Examples: Door slam, switch click, paper drop, footstep
[ ] No audio drift or delay
[ ] Transients are sharp and defined

KILL CONDITIONS:
- >2 frames of audio-visual drift
- Soft/mushy transients (lacks definition)
- Missing foley for visible actions
- Generic "stock sound" quality
```

**FOLEY AUTHENTICITY:**
```
[ ] Sounds match materials/surfaces shown
    Example: Paper on wood sounds different than paper on metal
[ ] Appropriate frequency content (not filtered incorrectly)
    Example: Footsteps have bass, paper rustle is high-frequency
[ ] Spatial positioning matches visual
    Example: Off-screen sounds are positioned off-axis

KILL CONDITIONS:
- Generic stock sounds that don't match materials
- All sounds center-panned (no spatial field)
- Missing bass frequencies on heavy objects
- "Canned" or library-obvious sounds
```

---

### 2.3 ENVIRONMENTAL AUDIO AUDIT

**ROOM TONE/AMBIENT BED:**
```
[ ] Continuous room tone appropriate to space
    - Concrete bunker: Short reverb, hard surfaces
    - Office: HVAC hum, fluorescent buzz, clock ticking
    - Studio: Equipment fans, tape deck motors
[ ] No dead silence (always some ambient)
[ ] Consistent throughout clip (no drops)

KILL CONDITIONS:
- Dead silence between sounds
- Inconsistent room tone (space changes)
- Wrong reverb for environment
- No HVAC or ambient machinery
```

**SPECIFIC ENVIRONMENTS:**
```
[ ] Broadcast Console: CRT electron hum, VU meters, tape deck
[ ] Document Close-up: Minimal ambient, paper focus
[ ] CRT Monitor: Loud electron hum -35dB, cooling fans
[ ] Surveillance: Heavy tape hiss, muffled/compressed foley
[ ] Corporate Office: HVAC prominent, fluorescent buzz
[ ] Underground Studio: Multiple equipment harmonics

KILL CONDITIONS:
- Wrong ambient for location
- Missing signature sounds for environment
```

---

### 2.4 AUDIO FREQUENCY AUDIT

**BANDWIDTH LIMITATION:**
```
[ ] High-pass filter at 80Hz applied
[ ] Low-pass filter at 12kHz applied
[ ] Simulates analog bandwidth limits
[ ] Appropriate frequency distribution per element

KILL CONDITIONS:
- Full 20Hz-20kHz range (too modern/digital)
- No HPF/LPF (sounds too clean)
- Sub-bass content below 80Hz (except 60Hz hum)
```

**FREQUENCY BALANCE:**
```
[ ] Low-end: 60Hz hum + bass foley appropriate
[ ] Midrange: Voice/main content clear but not hi-fi
[ ] High-end: Rolled off at 12kHz (analog character)
[ ] No digital harshness or sibilance

KILL CONDITIONS:
- Hyped high frequencies (digital sheen)
- Missing low-end weight
- "Broadcast perfect" frequency response
```

---

### 2.5 AUDIO LEVELING AUDIT

**MASTER LEVEL:**
```
[ ] Final level: -14 LUFS (YouTube) or -16 LUFS (TikTok/IG)
[ ] No clipping/distortion (except intentional)
[ ] Appropriate dynamic range maintained
[ ] Peak levels controlled

KILL CONDITIONS:
- Clipping/distortion present
- Too quiet (under -20 LUFS)
- Too loud/compressed (over -10 LUFS)
- Inconsistent levels between elements
```

### 3.2 TIMING/DURATION AUDIT

**DURATION COMPLIANCE:**
```
[ ] Clip is EXACTLY 10.0 seconds (if 10s format)
[ ] Beatmap timing aligns: H0.00 | B2.50 | B5.00 | B7.50 | E10.00
[ ] Actions complete within beats (not rushed or dragging)
[ ] Final hold is satisfying (not abrupt cut)

FAIL CONDITIONS:
- 9.8s or 10.3s (wrong duration)
- Actions misaligned with beatmap
- Rushed feeling (too much crammed)
- Abrupt ending (no terminal hold)
```

**PACING AUDIT:**
```
[ ] Opening hold establishes (2-3s)
[ ] Actions are deliberate, not frantic
[ ] Peak moment lands at appropriate time
[ ] Resolution/exit is clear
[ ] Overall pacing matches DEADMAN tone (measured, not rushed)

FAIL CONDITIONS:
- Too many actions (overcrowded)
- Frantic pacing (lacks control)
- No clear peak moment
- Unclear resolution
```

- AI/robotic voice artifacts

- No clear purpose/mission alignment

- Emotional/dramatic delivery
